
# EPUB Viewer

This repository hosts an **EPUB book viewer** using [EPUB.js](https://github.com/futurepress/epub.js) and GitHub Pages.

## How it works

- The EPUB file is served via GitHub Pages using a raw link.
- `index.html` contains the EPUB.js viewer to display the EPUB in a browser.
- You can embed the viewer in a form or website using an iframe.

## Example

GitHub Pages link: [https://abdulla227.github.io/epub-viewer/](https://abdulla227.github.io/epub-viewer/)

Embed in your site:

```html
<iframe src="https://abdulla227.github.io/epub-viewer/" width="100%" height="600px"></iframe>
